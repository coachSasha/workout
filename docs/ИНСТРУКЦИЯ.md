# Инструкция пользователя — Календарь тренера

Приложение для учёта тренировок: календарь записей, пакеты **соло / сплит / онлайн / бег**, история. Данные хранятся в Google Таблице (для вас это прозрачно — работа идёт через сайт).

---

## Кто чем пользуется

| Роль | Как зайти | Что можно делать |
|------|-----------|------------------|
| **Тренер** | Кнопка «Войти» → пароль | Календарь, клиенты, записи, выходные |
| **Гость** | Главная страница без входа | Только просмотр календаря и выходных |
| **Клиент** | Персональная ссылка `/c/...` | Остатки пакетов, ближайшие записи, история (без редактирования) |

---

## Вход и выход (тренер)

1. Откройте сайт приложения.
2. Нажмите **«Войти»** вверху справа.
3. Введите пароль тренера → **«Войти»**.
4. После входа появятся кнопки **«ЛК»** и **«Выйти»**.
5. **«Выйти»** — завершить сессию на этом устройстве (повторный вход снова по паролю).

> Сессия может сохраняться несколько дней в браузере, пока вы не нажмёте «Выйти».

## Календарь (тренер)

### Виды отображения

Вверху календаря: **Сегодня**, **Назад / Вперёд**, переключение **Месяц / Неделя / День**.

На телефоне по умолчанию удобнее режим **«День»**.

### Цвета записей

- **Соло** — свой цвет слота  
- **Сплит** — свой цвет  
- **Бег** — свой цвет  
- **Серым** — отменённые  
- **Зелёным** — проведённые (после подтверждения)  
- **Полоса «Выходной»** на весь день — день без тренировок  

### Выходной на день

1. Нажмите **«+ Выходной»** над календарём.  
2. Укажите дату и при желании заметку → **«Сохранить»**.  
3. В этот день **нельзя** создать новую запись.  
4. Чтобы убрать выходной: нажмите на полосу «Выходной» в календаре → **«Убрать выходной»**.

### Новая запись (соло / сплит / онлайн)

**На компьютере:** клик по свободному времени в календаре.  

**На телефоне:** **удерживайте** палец на свободной ячейке (~0,25 с), затем отпустите — откроется окно записи. Короткий тап нужен для прокрутки, не для записи.

В окне:

1. **Тип тренировки** — выберите **Соло** или **Сплит** (поле клиента станет активным).  
2. **Клиент** — выберите из списка (показываются только те, у кого есть остаток по этому типу).  
3. **«Сохранить»**.

В списке клиентов видны остатки: **С** — соло, **П** — сплит, **О** — онлайн, **Б** — бег.

### Новая запись (бег, несколько человек)

1. Откройте слот (клик или удержание на мобильном).  
2. Выберите тип **«Бег»**.  
3. Выберите время: **09:30** или **19:30**.  
4. В поле **«Клиенты»** откройте список (как обычный селект) и отметьте **одного или нескольких** участников.  
5. **«Сохранить»** — у каждого участника создаётся своя запись; в календаре они отображаются **одной** полосой «Бег» с именами.

### Просмотр и действия по записи

Нажмите на цветную полосу записи в календаре.

#### Соло / сплит / онлайн (один клиент)

| Кнопка | Действие |
|--------|----------|
| **Подтвердить (списать)** | Тренировка проведена → **−1** с пакета по типу слота |
| **Отменить со списанием** | Запись отменена, **−1** с пакета (как штраф за позднюю отмену) |
| **Отменить без списания** | Запись отменена, пакет **не** меняется |

#### Бег (группа)

Откроется карточка **«Бег — группа»** со списком участников.

- У каждого участника со статусом «Запланировано» — свои кнопки отмены **со списанием** / **без списания**.  
- **Частичная отмена:** отменили одного — остальные остаются в календаре, тренировка **не** считается полностью отменённой.  
- **Все отменены** — слот полностью свободен для переназначения.  
- **«Подтвердить всех пришедших»** — списывает **бег** у каждого, кто ещё в статусе «Запланировано» (уже отменённые не трогает).

### Переназначение отменённого слота

Если запись **полностью отменена** (один клиент или вся группа бега):

1. Откройте серую запись.  
2. **«Назначить другого клиента»** / **«Назначить на слот заново»**.  
3. Выберите клиента и тип → сохраните.

На один слот переназначение **один раз**: старая отменённая запись остаётся в истории, для слота создаётся новая.

> Пока на слоте висит непереназначенная отмена, нельзя создать другую запись кликом по пустому месту — сначала переназначьте через карточку отменённой записи.

## Личный кабинет (ЛК)

Кнопка **«ЛК»** в шапке (только после входа).

### Список клиентов

- Таблица: имя, фамилия, остатки по пакетам.  
- **«Добавить клиента»** — имя обязательно, фамилия и стартовые остатки по желанию.  
- Клик по строке — **карточка клиента**.

### Пополнение пакетов

В таблице — кнопки с **числом** у типов **Соло / Сплит / Бег** → укажите, сколько тренировок **добавить**.

### Карточка клиента

- Текущие остатки.  
- **Изменить остатки** — задать точное число в пакете (исправление ошибочного ввода).  
- **Удалить клиента** — удаляются все его тренировки (запланированные и из истории). Нельзя отменить.  
- **История тренировок** — дата, тип, статус.  
- **Ссылка для клиента** — скопируйте и отправьте клиенту.

### Удаление записи в календаре

В карточке тренировки (клик по полосе в календаре):

- **Удалить запись** — одна запись (соло/сплит/онлайн или один участник бега).  
- **Удалить всю запись (всех участников)** — вся группа бега на этот слот.  

Остатки пакетов при удалении **не меняются** (в отличие от отмены со списанием).

## Страница клиента (по ссылке)

Клиент открывает ссылку вида:  
`https://ваш-сайт/c/xxxxxxxx`  
**без пароля**.

На странице:

- остатки по пакетам;  
- ближайшие запланированные тренировки;  
- история с теми же статусами, что у тренера.

Клиент **не может** менять календарь и остатки.

---

## Просмотр без входа (гость)

На главной без входа:

- виден календарь с занятостью слотов (**без имён** клиентов — «Занято»);  
- видны выходные;  
- **нельзя** создавать, отменять и подтверждать записи.

---

## Пакеты и списания (кратко)

| Тип | Поле в системе | Когда списывается −1 |
|-----|----------------|----------------------|
| Соло | solo | Подтверждение или отмена **со списанием** |
| Сплит | split | То же |
| Онлайн | online | То же |
| Бег | running | Подтверждение группы / отмена участника **со списанием** |

- Записать клиента можно только если остаток по выбранному типу **> 0**.  
- **Отмена без списания** — остаток не меняется.  
- **Соло / Сплит / Онлайн**: длительность слота **60 минут**.  
- **Бег**: длительность слота **90 минут**, старт только **09:30** или **19:30**.  
- Пересечения по времени не допускаются: если на время уже есть запись (или отмена, которую нужно переназначить) — создать новую не получится.

## Частые вопросы

**Не могу записать на время — «На это время уже есть запись»**  
Слот занят активной записью или непереназначенной отменой. Откройте запись в календаре и переназначьте / отмените участников.

**На телефоне не открывается окно новой записи**  
Используйте **удержание** на ячейке, не короткий тап. Убедитесь, что вы **вошли** как тренер.

**После отмены одного из группы бега не подтверждается**  
Нажмите **«Подтвердить всех пришедших»** — спишется только у тех, кто ещё «Запланирован».

**Клиент не видит имя в календаре по ссылке**  
Так и задумано: в публичном календаре без входа имена скрыты. В персональной ссылке клиент видит **свои** записи.

**Кнопки «ЛК» нет**  
Нужно войти по паролю. Обновите страницу после входа.
